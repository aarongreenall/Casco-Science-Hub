B2.2 Size of specimens

Observing and measuring the size of biological specimens, particularly cells and organelles, is a crucial skill in biology. Since these structures are microscopic, we often use microscopes to magnify them and then apply specific formulas to calculate their actual size or the magnification of an image.

Learning Objectives:

  1. State and use the formula: magnification = image size $\div$ actual size.
  2. Calculate magnification and size of biological specimens using millimetres as units.
  3. Convert measurements between millimetres (mm) and micrometres ($\mu$m).

Magnification and Measuring Microscopic Specimens:

When observing tiny biological specimens under a microscope, the image we see is magnified. To determine the actual size of the specimen or the magnification of the image, we use the following relationship:

$Magnification = \frac{Image \ Size}{Actual \ Size}$

This formula can be rearranged to find any of the variables if the other two are known:

  • `Actual Size = Image Size / Magnification`
  • `Image Size = Magnification x Actual Size`

It's crucial to ensure that the **units for image size and actual size are consistent** (e.g., both in millimeters or both in micrometers) before performing calculations.

*Figure: Diagram illustrating the components of the magnification formula.*

Unit Conversions: Millimetres and Micrometres:

Biological specimens are often measured in very small units. You need to be able to convert between millimetres (mm) and micrometres ($\mu$m):

  • **1 millimetre (mm) = 1000 micrometres ($\mu$m)**
  • Therefore, to convert mm to $\mu$m, multiply by 1000.
  • To convert $\mu$m to mm, divide by 1000.

For example, if an image of a cell is 50 mm long and the magnification is x1000, its actual size would be: `Actual Size = 50 mm / 1000 = 0.05 mm` Or, in micrometres: `Actual Size = (50 mm * 1000 µm/mm) / 1000 = 50 µm`
