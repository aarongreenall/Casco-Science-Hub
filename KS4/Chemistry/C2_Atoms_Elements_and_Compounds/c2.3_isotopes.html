C2.3 Isotopes

While all atoms of a particular element have the same number of protons, they can differ in the number of neutrons in their nucleus. These variations are known as isotopes. Understanding isotopes is crucial for comprehending the diversity within elements and their applications in various fields, including medicine and dating.

Learning Objectives:

  1. Define isotopes as different atoms of the same element that have the same number of protons but different numbers of neutrons.
  2. State that isotopes of the same element have the same chemical properties because they have the same number of electrons and therefore the same electronic configuration.
  3. Interpret and use symbols for atoms, e.g. $^{12}_6C$ and ions, e.g. $^{35}_{17}Cl^-$.

Defining Isotopes:

**Isotopes** are atoms of the same element that have the **same number of protons** (and thus the same atomic number, $Z$) but **different numbers of neutrons** (and therefore different nucleon numbers, $A$).

For example, Hydrogen has three common isotopes:

  • **Protium ($^1_1H$):** 1 proton, 0 neutrons
  • **Deuterium ($^2_1H$):** 1 proton, 1 neutron
  • **Tritium ($^3_1H$):** 1 proton, 2 neutrons

All three are hydrogen because they all have 1 proton. They differ only in their mass due to the varying number of neutrons.

*Figure: Diagrams illustrating the three isotopes of hydrogen: Protium, Deuterium, and Tritium.*

Chemical Properties and Atomic Symbols:

Isotopes of the same element have the **same chemical properties**. This is because chemical properties are determined by the number of electrons in an atom and their arrangement in electron shells. Since isotopes of the same element have the same number of protons, a neutral atom of each isotope will also have the same number of electrons and thus the same electronic configuration.

Atoms and ions can be represented using **nuclide notation**: $^A_ZX$.

  • $A$ = Nucleon number (Mass number = protons + neutrons)
  • $Z$ = Proton number (Atomic number = number of protons)
  • $X$ = Chemical symbol of the element

For ions, the charge is added to the top right. For example, $^{35}_{17}Cl^-$ represents a chloride ion with a nucleon number of 35, 17 protons, and a -1 charge (meaning it has gained one electron).
