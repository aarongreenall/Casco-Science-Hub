P1.2 Motion

Motion is a fundamental concept in physics, describing how objects change their position over time. To accurately describe motion, we use specific terms like speed, velocity, and acceleration, each providing a different aspect of an object's movement.

Learning Objectives:

  1. Define speed as distance travelled per unit time; recall and use the equation $v = s/t$.
  2. Define velocity as speed in a given direction.
  3. Recall and use the equation average speed = total distance travelled / total time taken.
  4. Know that an object moving with increasing speed is accelerating, and that an object moving with decreasing speed is decelerating.
  5. Define acceleration as change in velocity per unit time; recall and use the equation $a = \Delta v / \Delta t$.
  6. Know that deceleration is a negative acceleration and use this in calculations.

Understanding Speed, Velocity, and Acceleration:

**Speed** tells us how fast an object is moving, without considering its direction. It is a scalar quantity, meaning it only has magnitude (size). The formula for speed is:

$Speed = \frac{Distance}{Time}$ or $v = \frac{s}{t}$

**Velocity** is similar to speed but includes the direction of motion. It is a vector quantity, meaning it has both magnitude and direction. For example, "70 km/h East" is a velocity, while "70 km/h" is a speed.

**Average speed** is calculated when an object's speed changes during a journey. It is the total distance covered divided by the total time taken for the journey.

$Average \ Speed = \frac{Total \ Distance \ Travelled}{Total \ Time \ Taken}$

**Acceleration** is the rate at which an object's velocity changes. This change can be an increase in speed, a decrease in speed (which is called **deceleration** or negative acceleration), or a change in direction. The formula for acceleration is:

$Acceleration = \frac{Change \ in \ Velocity}{Time \ Taken}$ or $a = \frac{\Delta v}{\Delta t}$

Understanding these distinctions is crucial for accurately describing and analyzing motion in physics.
